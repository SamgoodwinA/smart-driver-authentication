 🚗 Smart Driver Authentication System

 📌 Project Overview

The Smart Driver Authentication System is an AI-based solution designed to enhance road safety by identifying authorized drivers and detecting unsafe behaviors such as mobile phone usage while driving.

This system uses computer vision techniques to monitor driver activity in real time and trigger alerts when unsafe conditions are detected.

---

 🎯 Objectives

* To authenticate drivers using facial recognition
* To detect phone usage while driving
* To improve road safety through real-time alerts
* To provide an automated monitoring system

---

 🧠 Technologies Used

* Python
* OpenCV
* Machine Learning / Deep Learning
* NumPy
* SMTP (for email alerts)

---

 ⚙️ System Features

* ✅ Driver face recognition
* ✅ Phone usage detection
* ✅ Real-time alert system (audio warning)
* ✅ Email notification for unauthorized driver

---

 🏗️ System Architecture

1. Capture video input using camera
2. Detect face and identify driver
3. Monitor hand/phone usage
4. Trigger alerts if unsafe behavior detected
5. Send email notification if unknown driver detected

---

 ▶️ How to Run the Project

```bash
pip install -r requirements.txt
python main.py
```
